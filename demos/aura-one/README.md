# Aura One — 애플 제품 페이지 문법 연구

**가상 브랜드** 무선 이어버드. 실제 로고·제품명·카피·사진을 일절 쓰지 않고, 디자인 "문법"만 연구했습니다.

## 연구한 문법
- **스크롤이 곧 프레젠테이션**: 히어로 → sticky 스테이지에서 제품이 스크롤에 따라 회전·분해·재조립 → 기능 카피가 한 문장씩 낙하.
- **거대 타이포 + 여백**: 화면을 가르는 제목 스케일, 넉넉한 상하 여백, 중앙 정렬.
- **타이포 스케일**: 히어로 96~128px, 기능 카피 28~56px, 스펙 숫자 강조.

## 어떻게 재현했나
- 제품은 **인라인 SVG**(바이너리 에셋 0). 분해/회전은 **CSS Scroll-driven Animations**(`animation-timeline: scroll()/view()`, `animation-range`). 기능 카피 낙하도 `view()`.
- 미지원 브라우저는 `@supports`로 감지 → 내용 정적 노출(효과만 생략).
- 색·서체는 시그니처를 피한 자체 그래파이트·뉴트럴 팔레트 + 시스템 폰트.

## 브라우저 / 실행
Scroll-driven Animations 지원 브라우저 권장. `index.html` 열기.
