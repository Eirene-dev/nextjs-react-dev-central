# Aura One — 애플 제품 페이지 문법 연구

**가상 브랜드** 무선 이어버드. 실제 로고·제품명·카피·사진을 일절 쓰지 않고, 디자인 "문법"만 연구했습니다.

## 연구한 문법
- **길게 이어지는 스크롤 프레젠테이션**: 히어로(단어 순차 리빌) → 핀 시퀀스 3종(이어버드 회전·분해 → 노이즈 캔슬링 음파 소거 → 공간 음향 음장 회전) → 기능 카피 낙하 → 스펙 → 가격. 각 핀은 sticky 스테이지 + 360vh.
- **전역 스크롤 메커니즘**: 상단 진행 바(`scroll(root)`), 섹션 명암 리듬(라이트 ↔ 딥다크, 다크 섹션 텍스트 반전).
- **거대 타이포 + 여백**: 화면을 가르는 제목 스케일, 넉넉한 상하 여백, 중앙 정렬.
- **타이포 스케일**: 히어로 56~128px, 기능 카피 28~56px, 스펙·−42dB 숫자 강조.

## 어떻게 재현했나
- 제품·음파·음장은 **인라인 SVG**(바이너리 에셋 0). 회전·분해·음파 소거·음장 회전은 **CSS Scroll-driven Animations**(요소별 `animation-timeline: view()`, 전역 진행 바는 `scroll(root)`, `animation-range`로 구간). 기능 카피 낙하도 `view()`. **JS 0**(히어로 단어 리빌은 로드 stagger, 진행도 효과는 전부 scroll-timeline).
- 미지원 브라우저는 모든 신규 효과에 `@supports`로 감지 → 핀 `height:auto`로 내용 정적 노출(효과만 생략, 깨지지 않음).
- 색·서체는 시그니처를 피한 자체 그래파이트·뉴트럴 팔레트 + 시스템 폰트.

## 브라우저 / 실행
Scroll-driven Animations 지원 브라우저 권장. `index.html` 열기.
