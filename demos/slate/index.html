Slate — JS 없는 UI 레이어

오버레이·폼·내비 UI 13종 — 툴팁·드롭다운·중첩·컨텍스트·다이얼로그·아코디언·탭·토글·드로어·인보커·커스텀 셀렉트·캐러셀·토스트. 전부 Popover API + CSS Anchor Positioning + @starting-style + <details> + :checked + command + base-select로.

이 브라우저는 Popover API(:popover-open)를 지원하지 않습니다 — 버튼은 보이나 팝오버가 열리지 않을 수 있습니다.
이 브라우저는 CSS Anchor Positioning을 지원하지 않습니다 — 레이어 위치가 어긋날 수 있습니다.

아코디언 (Accordion)

<details> + interpolate-size 높이 애니. 한 번에 하나만(name).

JS가 정말 0줄인가요?
네 — 인터랙션 스크립트가 0줄입니다. 펼침/접힘은 <details>의 브라우저 기본 동작입니다.
높이 애니메이션은 어떻게?
interpolate-size: allow-keywords + ::details-content의 block-size transition. 미지원 시 즉시 토글로 폴백.
접근성은요?
네이티브 요소라 키보드·스크린리더가 기본 지원됩니다.
