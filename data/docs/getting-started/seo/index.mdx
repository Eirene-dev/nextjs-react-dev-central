## 검색 엔진 최적화 학습 가이드

이 학습 가이드에서는 검색 엔진 최적화(SEO)의 중요성부터, Next.js 프로젝트에서 SEO를 효과적으로 구현하고 모니터링하는 방법까지 상세하게 다룹니다. 각 섹션은 이론과 실제 구현 예제로 구성되어 있어, 초보자부터 전문가까지 모두 SEO에 대한 통찰력을 얻을 수 있습니다.

<Card href="/docs/getting-started/seo/introduction-to-seo">
### SEO 소개

EO의 정의와 중요성, 최적화의 핵심 세 가지 기둥, 검색 시스템의 원리, 웹 크롤러의 역할, 그리고 Googlebot의 작동 방식에 대해 설명합니다.
</Card>

<Card href="/docs/getting-started/seo/crawling-and-indexing">
### 크롤링 및 색인화

웹사이트의 검색 엔진 최적화를 위한 기본적인 크롤링 및 인덱싱 방법, robots.txt와 XML 사이트맵 설정, 그리고 특별한 메타 태그와 캐노니컬 태그의 사용법을 Next.js 프로젝트 관점에서 소개합니다.
</Card>

<Card href="/docs/getting-started/seo/rendering-and-ranking">
### 렌더링 및 랭킹

다양한 렌더링 전략, Next.js의 경로 정의, Open Graph, 구조화된 데이터, 그리고 페이지 내 SEO에 관한 주요 내용을 탐구합니다.
</Card>

<Card href="/docs/getting-started/seo/web-performance">
### 웹 성능 & 코어 웹 바이탈

Google이 웹 사용자 경험을 측정하기 위해 제시한 핵심 웹 척도(LCP, FID, CLS)에 대한 개념, SEO와 UX에 미치는 영향, 그리고 Lighthouse의 척도 가중치에 관한 정보를 제공합니다.
</Card>

<Card href="/docs/getting-started/seo/improve">
### 코어 웹 바이탈 개선

Lighthouse 도구를 통한 웹 성능 척도와 이미지, 폰트, 스크립트 등의 최적화 방법 및 예제를 중점적으로 다룹니다.
</Card>

<Card href="/docs/getting-started/seo/monitor">
### 코어 웹 바이탈 모니터링

Next.js를 활용한 성능 모니터링 방법, 사용자 정의 보고 설정, 크롬 사용자 경험 보고서를 이용한 분석, 그리고 다양한 성능 측정 도구에 대해 소개합니다.
</Card>
