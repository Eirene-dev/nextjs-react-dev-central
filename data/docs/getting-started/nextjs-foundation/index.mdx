## Next.js 학습 가이드

이제 여러분은 웹 프론트엔드의 미래로 불리는 Next.js를 학습하려는 중요한 기회를 얻게 되었습니다. 세계에서 가장 선도적인 기업들은 사용자 중심의 경험을 제공하기 위해 Next.js를 선택하고 있습니다. 그렇기 때문에 이 기술을 습득하는 것은 여러분의 개발 경력에 있어서 큰 전환점이 될 수 있습니다. 단순히 코드를 작성하는 것이 아닌, 성능, 접근성, 그리고 사용자 경험에 중점을 둔 웹 애플리케이션을 구축하는 능력을 키울 수 있는 시기입니다. 이 가이드는 그 첫 걸음을 돕기 위해 준비되었습니다. 

<div className="grid grid-cols-1 sm:grid-cols-2 gap-4">

<Card href="/docs/getting-started/nextjs-foundation/about-nextjs">
### Next.js 개요

Next.js와 웹 애플리케이션의 구성 요소를 탐구하며, React와의 연관성도 함께 알아봅니다.
</Card>

<Card href="/docs/getting-started/nextjs-foundation/from-javascript-to-react">
### JavaScript에서 React로

JavaScript의 기본부터 시작하여 DOM의 이해, UI 렌더링 방법, 그리고 React의 핵심 개념과 JSX에 이르기까지 웹 프론트엔드 개발에 필요한 중요한 주제들을 포괄적으로 다룹니다. 
</Card>

<Card href="/docs/getting-started/nextjs-foundation/from-react-to-nextjs">
### React에서 Next.js로

React 코드를 Next.js 앱으로 마이그레이션 후, 개발 서버 실행 하는 법을 다룹니다.
</Card>

<Card href="/docs/getting-started/nextjs-foundation/how-nextjs-works">
### Next.js 동작 방식

개발 환경에서 운영 환경으로 전환하는 과정, Next.js의 컴파일링, 미니파이, 번들링, 코드 스플리팅 같은 핵심 컴파일러 기술들을 소개하고, 어떻게 클라이언트와 서버 간에 렌더링이 이루어지는지, 그리고 네트워크의 다양한 구성 요소와 그들의 역할에 대해 설명합니다.
</Card>

</div>

### 'v13.4(\'23.05): 앱 라우터

Next.js 13.4 릴리즈에서는 앱 라우터를 공식화하며, 설정 없이 파일시스템을 API로 사용하고, 모든 것이 함수 기반의 자바스크립트로 제공되며, 자동 서버 렌더링 및 코드 분할 기능을 포함하고 있습니다.
</Card>
