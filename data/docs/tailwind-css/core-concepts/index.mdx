## 핵심 개념

전통적인 CSS 작성과 다른 테일윈드 CSS의 핵심 개념을 살펴보세요.
<div className="grid grid-cols-1 sm:grid-cols-2 gap-4">

<Card href="/docs/tailwind-css/core-concepts/utility-first">
### 유틸리티 중심
유틸리티 중심 워크플로우를 사용하여 제한된 기본 유틸리티 세트로 복잡한 구성 요소를 구축합니다.
</Card>

<Card href="/docs/tailwind-css/core-concepts/responsive-design">
### 반응형 디자인
반응형 수식어를 사용하여 모든 화면 크기에 적응하는 완전 반응형 사용자 인터페이스를 구축합니다.
</Card>

<Card href="/docs/tailwind-css/core-concepts/hover-focus-and-other-states">
### 호버, 포커스 및 기타 상태
호버, 포커스 등의 상호작용 상태에서 요소를 스타일링하는 조건부 수정자를 사용합니다.
</Card>

<Card href="/docs/tailwind-css/core-concepts/dark-mode">
### 다크 모드
HTML에서 바로 다크 모드 수정자를 사용하여 사이트를 다크 모드에 최적화하세요.
</Card>

<Card href="/docs/tailwind-css/core-concepts/reusing-styles">
### 스타일 재사용
재사용 가능한 추상화를 생성하여 중복을 관리하고 프로젝트의 유지 관리 가능성을 유지하세요.
</Card>

<Card href="/docs/tailwind-css/core-concepts/adding-custom-styles">
### 프레임워크 사용자 정의
브랜드와 일치하도록 프레임워크를 사용자 정의하고 자체 사용자 정의 스타일로 확장하세요.
</Card>

<Card href="/docs/tailwind-css/core-concepts/functions-and-directives">
### 함수와 지시어
함수와 지시어를 사용하여 Tailwind CSS를 통해 스타일을 보다 쉽게 관리하고 사용자 정의할 수 있습니다.
</Card>

</div>
